# sieben-server

## Clone
```
$ mkdir -p ~/go/src/github.com
$ cd ~/go/src/github.com
$ git clone ...
$ cd sieben-server
```

## Install Dependencies
```
$ make deps
```

## Build (OSX)
```
$ make build_osx
```

## Test
```
$ make test
```

## Run (OSX)
```
$ ./bin/osx/sieben-server
```

## Clean
```
$ make clean
```
